# Changelog

Все значимые изменения в этом проекте будут документироваться в этом файле.

Формат основан на [Keep a Changelog](https://keepachangelog.com/ru/1.0.0/),
и этот проект придерживается [Semantic Versioning](https://semver.org/lang/ru/).

## [Unreleased]

### Added
- Git алиасы для быстрой работы
- Шаблоны для Issues и Pull Requests
- GitHub Actions CI/CD
- Pre-commit hooks для проверки кода
- EditorConfig для единообразия кода
- Документация по работе с Git
- Git helpers - полезные функции для работы с Git
- Changelog для отслеживания изменений
- Локальная конфигурация Git для проекта

### Changed
- Улучшена структура коммитов
- Оптимизирована работа с Git
- Улучшена мобильная версия портфеля
- Оптимизированы блоки Топ-5 для мобильных устройств

### Fixed
- Исправлен формат даты в git-aliases для macOS
- Исправлено переполнение текста в карточках на мобильных
- Исправлена проблема с отображением блоков одинакового размера
- Убрана автоматическая перезагрузка страницы

## [1.0.0] - 2024-12-08

### Added
- Инициализация проекта в Git
- README.md с описанием проекта
- Wiki шаблоны для документации
- Скрипты для автоматизации работы с Git
- Карусель со свайпом для блоков Топ-5
- Кнопка "Показать все" для просмотра всех проектов
- Адаптивный дизайн для мобильных устройств
- Pull-to-refresh функциональность (позже удалена)
- SwipeableCard компонент для свайпа карточек
- FormStepper для многошаговых форм
- StickyActions для закрепленных кнопок действий

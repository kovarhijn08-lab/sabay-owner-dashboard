📋 ЧТО ЭТО ЗА ПРОЕКТ?
─────────────────────────────────────────────────────────────────
Личный кабинет для инвесторов Sabay Property, купивших 
недвижимость на Пхукете.

Основной функционал:
  • Управление портфелем объектов недвижимости
  • Отслеживание прогресса строительства
  • Мониторинг доходности аренды
  • Сравнение с рынком
  • Уведомления и алерты

🏗️ ТЕХНОЛОГИИ
─────────────────────────────────────────────────────────────────
Backend:  NestJS, TypeORM, PostgreSQL
Frontend: Next.js 14, React, Tailwind CSS
Auth:     JWT (Passport)
Язык:     TypeScript

📁 СТРУКТУРА
─────────────────────────────────────────────────────────────────

apps/dashboard/  → Next.js Frontend (порт 3001)

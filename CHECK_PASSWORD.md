# ✅ Проверка пароля через pgAdmin

## Проверьте, что пароль установлен правильно

### Способ 1: Проверка через SQL в pgAdmin

1. В pgAdmin нажмите вкладку **"SQL"** (вверху)

2. Введите команду для проверки:
   ```sql
   SELECT current_user;
   ```

3. Нажмите **Execute** (▶️) или F5

4. Должно показать: `postgres`

### Способ 2: Попробуйте установить пароль ещё раз

1. В pgAdmin нажмите вкладку **"SQL"**

2. Введите команду:
   ```sql
   ALTER USER postgres WITH PASSWORD 'a7NGEr2k!';
   ```

3. Нажмите **Execute** (▶️) или F5

4. Должно появиться сообщение: `ALTER ROLE`

### Способ 3: Проверьте подключение

После установки пароля попробуйте переподключиться в pgAdmin:
- Отключитесь от сервера (правый клик → Disconnect Server)
- Подключитесь снова с паролем `a7NGEr2k!`

Если подключение работает - пароль правильный!

# ❓ Часто задаваемые вопросы (FAQ)

## Общие вопросы

### Что такое Sabay Owner Dashboard?
Это веб-приложение для управления портфелем недвижимости, которое позволяет владельцам отслеживать свои объекты, анализировать доходность и управлять финансами.

### Какие технологии используются?
- **Frontend**: Next.js 14, TypeScript, Tailwind CSS
- **Backend**: NestJS, TypeORM
- **База данных**: PostgreSQL (продакшен) или SQLite (разработка)

## Разработка

### Как запустить проект в режиме разработки?
```bash
pnpm dev
```

### Как создать нового пользователя?
```bash
cd apps/api
pnpm run create:owner  # Для владельца
pnpm run create:admin  # Для администратора
```

### Как добавить тестовые данные?
```bash
cd apps/api
pnpm run add:test-data
```

## Использование

### Как войти в систему?
1. Откройте http://localhost:3001
2. Введите email и пароль
3. Нажмите "Войти"

### Как добавить новый объект недвижимости?
1. Перейдите в "Портфель" → "Объекты"
2. Нажмите "Добавить объект"
3. Заполните форму
4. Сохраните

### Как просмотреть финансовые метрики?
1. Перейдите в "Портфель"
2. Выберите объект
3. Откройте вкладку "Финансы"

## Проблемы

### Проект не запускается
1. Проверьте версию Node.js: `node --version`
2. Убедитесь, что зависимости установлены: `pnpm install`
3. Проверьте логи в консоли

### Ошибка подключения к базе данных
1. Проверьте настройки в `.env`
2. Убедитесь, что база данных запущена
3. Проверьте права доступа

### Страница не загружается
1. Проверьте, что сервер запущен
2. Проверьте консоль браузера на ошибки
3. Проверьте логи сервера

### Забыл пароль
Обратитесь к администратору для сброса пароля.
